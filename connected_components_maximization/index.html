TP N°4 : Les points d'articulations dans un graphe non-orienté

Le problème consiste à trouver les points d'articulation dans un graphe non orienté.
Ce problème est un grand classique en algorithmique et il existe diverses méthodes pour le résoudre.
Dans le cadre de ce TP, on est amené à utiliser la méthode dite le parcourt DFS des graphs pour résoudre ce problème.
